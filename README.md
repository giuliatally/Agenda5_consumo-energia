# ⚡ Calculadora de Consumo de Energia ⚡

## 📍 Objetivo 📍
Este projeto, demandado na agenda 5 do curso de desenvolvimento de sistemas da ETEC, tem como objetivo calcular o consumo mensal de energia elétrica de um aparelho (em watts e em reais), com base na sua potência e no tempo de uso diário.

## ⁉️ Fórmula para o Cálculo
As fórmulas utilizadas para o cálculo são:<br>
1. consumo_mensal = (potencia * horas_dia * 30) / 1000<br>
2. custo = consumo_mensal * valor_kwh

## 📓 Instruções de Uso
Para utilizar a calculadora, siga os passos abaixo:
1. Clonar esse repositório: [Calculadora de Consumo de Energia](https://github.com/giuliatally/Agenda5_consumo-energia)
2. Acessar a pasta do projeto: "cd Agenda5_consumo-energia"
3. Executar o programa: python app.py
4. Digitar o nome do aparelho, sua potência em watts e a média de horas de uso por dia no terminal, conforme as perguntas feitas a você
